AI-integrated ops tools

Four small, working apps built to practice the patterns behind real enterprise tooling — a SOC console, an ITSM helpdesk, a sales CRM, and an identity & access directory. Each one makes a live call to Claude through a rate-limited proxy for a specific AI-assisted workflow. Built with Claude Code.

◆ SOC-CONSOLE
Detection & Triage Console
Simulated SIEM alert queue. AI triage classifies severity and writes an analyst-style explanation and next step for any alert.
◆ HELPDESK
ITSM Ticket Queue
Ticket queue with SLA tracking. AI triage auto-categorizes, sets priority, and drafts a first-response reply.
◆ PIPELINE
Sales Pipeline CRM
Kanban deal pipeline with activity notes. AI summarizes deal history and recommends a concrete next action.
◆ DIRECTORY
Identity & Access Directory
Simulated AD directory with RADIUS and TACACS+ event history per user. AI scans access patterns to flag likely-compromised accounts.
